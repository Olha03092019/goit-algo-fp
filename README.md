Висновки до Завдання 7

У ході виконання завдання було реалізовано метод Монте-Карло для моделювання
кидків двох ігрових кубиків та обчислення ймовірностей появи кожної можливої суми.

Порівняння результатів симуляції Монте-Карло з аналітичними розрахунками показало, що:
- при малій кількості кидків результати можуть суттєво відрізнятися від теоретичних
- зі збільшенням кількості симуляцій емпіричні ймовірності поступово
  наближаються до аналітичних значень
- найбільшу ймовірність має сума 7, а найменшу - суми 2 та 12, що відповідає
  теоретичним розрахункам

Метод Монте-Карло коректно відтворює розподіл імовірностей для задачі кидання двох кубиків.
